# ろるさぽくん — 要件書

## このプロダクトが解決する問題

LoL（League of Legends）は判断の連続で成り立つゲームだが、初中級者は**「今何をすべきか」がわからない**。

- **ビルド**: 敵構成に応じたアイテム選択が難しい。ガイドサイトを見ても「この試合」に最適かはわからない
- **対面知識**: 100体以上のチャンピオンとのマッチアップを全て覚えるのは非現実的。知らない対面で何に気をつけるべきかわからない
- **マクロ判断**: オブジェクトの優先順位、いつ集合すべきか、いつファームすべきか。試合中にリアルタイムで考えるのは難しい
- **振り返り**: 試合後に「何が悪かったのか」を客観的に分析する手段が少ない

既存のガイドサイト（OP.GG等）は統計データを提供するが、**今この瞬間の試合状況に応じたアドバイス**はくれない。

## プロダクトのアプローチ

**試合中にリアルタイムで寄り添うAIコーチ**を提供する。

LoLクライアントのAPIからゲームデータを自動取得し、AIがその場の状況に応じたアドバイスを画面上に常時表示する。プレイヤーは手を止めずにチラ見するだけで次のアクションの指針を得られる。

### コア機能

#### 1. アイテム提案
**問題**: 敵構成やゲーム展開に応じたアイテム選択が難しい

**アプローチ**: OP.GGの統計ビルドをベースに、AIが「この試合では何を変えるべきか」を提案する。敵にヒーラーが多ければ回復阻害、敵ADCが育っていれば防具、のように文脈を踏まえた判断を返す。

**理想の体験**: 「コアビルドはこれだけど、この試合は敵のAP3人構成だからMR積んだほうがいい」とコーチが横で教えてくれる感覚

#### 2. 対面マッチアップTip
**問題**: 知らない対面に当たると何に気をつけるべきかわからない

**アプローチ**: チャンプセレクト完了時に、自分と対面チャンピオンの相性・レーニングのコツ・パワースパイク・危険タイミングを提示する。

**理想の体験**: 試合が始まる前に「この対面はLv6以降が危ないから、それまでに有利を作っておこう」と心構えができる

#### 3. マクロアドバイス
**問題**: ゲーム中盤以降、どこにいて何をすべきかの判断が難しい

**アプローチ**: オブジェクトのスポーン状況・チームのキル差・ゲーム時間・ポジションを総合的に見て「今すべき最善のアクション1つ」を具体的に指示する。抽象的な「マップを見ろ」ではなく、「ドラゴンが1分後にスポーンするからボットサイドにワードを置いて、ウェーブをプッシュしておく」のような手順付きのアドバイス。

**理想の体験**: マップを見て判断できる上級者の思考プロセスを、リアルタイムで補助してもらえる

#### 4. ルールベースアラート
**問題**: ファーム効率の低下やオブジェクトタイミングを見逃す

**アプローチ**: AIに頼らない即時判定で、CS低下・オブジェクトスポーン・デス頻度などをアラート通知。AIの応答を待たずに瞬時にフィードバックする。

**理想の体験**: 「CSが落ちてきてるぞ」「ドラゴンまであと1分、準備しろ」と背後から声をかけてくれるコーチ

#### 5. 試合後コーチング
**問題**: 試合が終わっても何が良くて何が悪かったのかわからない

**アプローチ**: 試合全体のデータを分析し、各カテゴリ（レーン戦・集団戦・ファーム・ビジョンなど）をグレード評価。良かった点と改善点を具体的に挙げ、次の試合に向けたアドバイスを提供する。

**理想の体験**: 対戦後レビューを毎試合やってくれるコーチがいる感覚

## 大切にすること

- **試合体験を邪魔しない**: コンパクトなオーバーレイ。チラ見でわかる簡潔さ。ゲーム中にウィンドウ切り替え不要
- **具体的であること**: 「マップを見ろ」ではなく「ドラゴン1分前、ボットにワードを置いてウェーブをプッシュ」。行動に直結するアドバイス
- **今の試合に合っていること**: 統計データの丸写しではなく、目の前の敵構成・キル差・ゲーム展開を踏まえた判断

## ゴール

ユーザーの真横にプロのコーチがいるかのような体験を提供すること。
