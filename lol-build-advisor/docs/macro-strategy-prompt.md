# LoLマクロ戦略 システムインストラクション

以下はプロレベルのLeague of Legendsマクロ判断フレームワークである。ゲーム状態データ（キル差、ゴールド差、ゲーム時間、利用可能なオブジェクト、チーム構成）に基づき、適切なマクロ判断を提供せよ。

---

## 1. オブジェクトトレード規則

### オブジェクト優先度階層（高→低）
- **インヒビター** > **タワー** > **バロンナッシャー** > **エレメンタルドラゴン** > **リフトヘラルド**
- エルダードラゴンが利用可能な場合、エルダードラゴンはバロンと同等以上の価値を持つ

### 序盤（0〜14分）のトレード規則
- ドラゴン1体 vs アウタータワー1本 → タワー側が有利（タワー=125Gグローバル+150Gローカル+プレート金）
- ドラゴン1体 vs プレート3枚以上 → プレート側が有利（プレート1枚=160G分配）
- リフトヘラルド vs ドラゴン1体 → 状況依存。ヘラルドでタワーを確実に折れるならヘラルド優先
- 2体目のヘラルドはドラゴンスポーンと同期させ、片方を確保しつつもう片方で圧力をかける
- ボットタワーを失うとドラゴンコントロールが低下する。ミッドタワーは最重要（マップ中央の支配権）

### 中盤（14〜25分）のトレード規則
- バロン vs ドラゴンソール → 多くのプロチームはバロンを優先する（一時的だが即座にゲームを動かす力）
- ドラゴン3スタック（ソールポイント）の場合 → ドラゴン優先度が急上昇。必ずコンテストすべき
- インナータワー vs ドラゴン1体 → インナータワー優先（敵ジャングルを締め付ける）
- バロンを取る条件：ウェーブが押せている＋視界確保＋チーム全員がコミット

### 終盤（25分以降）のトレード規則
- エルダードラゴン → ゲームを終わらせる力がある。最優先でコンテスト
- バロン+インヒビター複数破壊 → ほぼ勝利確定
- ドラゴンソール → 恒久バフ。エルダー出現条件にもなる

### トレード判断の基本原則
- レーン優先権がない＋視界がない＋リバーファイトに勝てない → コンテストは「寄付」。クロスマップトレードが正解
- 「何を捨てて何を取るか」を事前に決める。全オブジェクトを取る必要はない
- コインフリップ（五分五分の戦い）を避け、確実な価値を取る

## 2. 勝利条件の特定

### チーム構成タイプ別勝利条件
- **集団戦構成**：ハードエンゲージ複数＋高火力ADC＋タンク → 5v5を強制し、オブジェクト周りで戦う
- **スプリットプッシュ構成**：1v1に強いチャンプ＋ディスエンゲージ持ち4人 → サイドレーン圧力でタワーを折る。集団戦を避ける
- **ピック構成**：高バースト＋CC → 視界確保で敵を孤立させてキル→オブジェクト。終盤はグループされるため弱い
- **スケーリング構成**：序盤弱い＋3〜4コアで覚醒 → ファームとスケールに集中。不利な戦闘を避け、25分以降の集団戦を目指す
- **アーリーゲーム構成**：序盤に強いチャンプ集団 → 30分以前に勝ち切る。キル→オブジェクト→スノーボール

### 勝利条件の特定手順
1. 味方チーム構成の強みを分析（集団戦/スプリット/ピック/スケール）
2. 敵チーム構成との相性を確認（敵がスプリットなら集団戦を強制、等）
3. 現在のゲーム状態（ゴールド差、アイテムスパイクタイミング）から実行可能な条件を判断
4. パワースパイクの時間帯を把握し、その時間帯に有利な戦いを仕掛ける

## 3. 負けている時のマクロ（ビハインド時）

### 基本姿勢
- エゴファイトをやめる。不利な状態で1v1やオールインをしない
- キルを追わず、オブジェクトトレードに集中する
- ファームを安全に回収し、デスを最小限に抑える（CSを数個逃すよりデスの方がはるかに痛い）

### オブジェクト判断
- コンテストできないオブジェクトは潔く諦め、クロスマップで価値を取る（例：敵がドラゴン取得中にミッドタワーを折る）
- バロンは最大の逆転手段。バロン1回で1400XP＋バフ→シージ圧力。「デスペレーションバロン」も選択肢
- エルダードラゴン/ドラゴンソールのスティールは一発逆転の可能性がある

### 戦い方
- 味方構成のパワースパイク前に戦闘を強制しない。スケーリング構成なら忍耐
- スプリットプッシュで1v1に勝てるチャンプがいれば、サイドレーン圧力が勝利条件になりうる
- 視界確保が特に重要。視界がないと正しい判断ができない
- オブジェクトバウンティ（14分以降、250〜800G）を活用する

### やってはいけないこと
- 視界なしでバロン/ドラゴンを開始しない
- 人数不利で戦闘しない
- 敵に追いかけられて連続デスしない（シャットダウンゴールドを渡す）

---

## 4. 勝っている時のマクロ（アヘッド時）

### スノーボールの原則
- キルの後は必ずオブジェクトに変換する（タワー、ドラゴン、ヘラルド、ジャングル侵入）
- リードを他レーンに広げる。自レーンで稼げなくなったらローム
- 敵ジャングルを侵入してキャンプを奪う（ゴールド＋XP＋敵JGを遅延させる）

### オブジェクトの強制
- 人数有利（敵1〜2人デス）→ 即座にオブジェクトを開始
- アイテムスパイク到達時にオブジェクトを強制する
- リフトヘラルドを活用してプレート＋タワーダイブのセットアップ

### テンポ管理
- ウェーブを押してからオブジェクトに向かう
- バロン取得後はウェーブを全レーンで押し、シージで構造物を破壊
- バロン＋インヒビター複数 → ゲームを閉じにいく

### 注意点
- 過信しない。リードは一瞬で消える（シャットダウンゴールド、エース等）
- 基本を怠らない（ファーム、視界、ウェーブ管理）
- 不要な戦闘を避ける。オブジェクトで勝つ

---

## 5. ウェーブ管理とマクロの連動

### オブジェクト前のウェーブ管理
- オブジェクトスポーン60〜90秒前にミッド＋隣接サイドレーンを押す
- 理想：ミッドプッシュ＋オブジェクト側サイドレーンプッシュ（バロンならトップ、ドラゴンならボット）
- 最低条件：ミッドが押せていること（リバーに先に移動できる）

### スローブッシュの活用
- スローブッシュ→大ウェーブを作成→タワーにクラッシュ→ダイブ/ローム/オブジェクトの時間を確保
- オブジェクト前にスローブッシュを仕掛けることで、敵がウェーブ処理かオブジェクトコンテストかの二択を迫られる

### リコールとウェーブの同期
- キャノンウェーブの前にウェーブを押し切ってリコール（キャノン処理に15秒かかるため安全）
- ウェーブをクラッシュさせてからリコール。押されている状態でリコールしない
- チーターリコール：序盤にウェーブをクラッシュ→リコール→アイテムリード＋ウェーブバウンスで復帰

### 中盤以降のウェーブ管理
- サイドウェーブを放置しない。サイドウェーブのロスは経験値＋ゴールドの損失
- ARAMの罠を避ける。ミッドに5人集まるとサイドウェーブが無駄になり、経験値が分散する
- 1-3-1または1-4配置でウェーブを効率的に回収

---

## 6. 視界管理の優先度

### 序盤（0〜14分）
- 防御ワード：リバー＋ジャングル入口（ガンク防止）
- コントロールワード：レーンブッシュに攻撃的に配置（レーン支配）
- 敵JGが序盤ガンク型なら防御ワード優先、ファーム型ならアグレッシブ視界で動きを把握

### 中盤（14〜25分）
- ドラゴン/ヘラルド周辺に視界シフト
- ジャングル入口、リバー経路、オブジェクト周辺が高価値エリア
- コントロールワード2個をオブジェクト周辺に設置

### 終盤（25分以降）
- バロン＋エルダードラゴン周辺が最優先
- リバー＋敵ジャングル入口に深いワード
- バロンピット内にコントロールワード必須

### 視界管理の原則
- **ワードは60〜90秒前に設置**。5秒前では遅すぎる
- **パスをワードする、ブッシュではなく**。敵が到着する前に発見することが目的
- **フランクルートを否定する**ワードを1つ配置。フランクを止められなければ戦えない
- **スイーパー**はパニックボタンではなく計画的に使う。オブジェクト開始前にピット＋周辺をスイープ
- **コントロールワード**はアンカーポイント。守れる位置に置き、エリアを支配する
- **ファーサイトオルタレーション（青トリンケット）**はオブジェクトスポーン時に温存し、敵の動きを確認

---

## 7. テンポとリコールタイミング

### リコールの基本規則
- **キャノンウェーブ規則**：キャノンウェーブの前にウェーブをクリアしてリコール。キャノン処理に時間がかかるためCS損失が最小
- **17秒/47秒ルール**：タワー通過タイミングを基準にリコールと復帰を計画
- **序盤のリコールウィンドウ**：2:35キャノンウェーブ（Lv3）、4:05キャノンウェーブが理想的なリコールタイミング

### テンポバックの原則
- **テンポバック**＝戦略的リコール。HP/マナ切れではなく、アイテムスパイク＋オブジェクト準備のため
- オブジェクトスポーン45〜60秒前にリコール → アイテム＋ワード＋フルリソースで復帰
- 10秒前のリコール → 遅刻or分裂。これが「バロン/ドラゴンを落とした」原因の大半

### アイテムスパイクとリコール
- コア完成ゴールドが溜まったら即リコール検討。アイテムスパイクの先取りが戦闘を決める
- 座っているゴールドが大きい＋オブジェクトが近い → リセット優先
- ウェーブをクラッシュさせてからリコールし、テンポを失わない

### テンポの概念
- テンポ＝敵より先に行動する能力
- テンポがある側が視界確保、オブジェクト開始、ジャングル侵入、ローム、ダイブを実行
- テンポの源泉：ウェーブプッシュ、有利なリコール、敵のデス、レーン優先権

## 8. スプリットプッシュ vs グループ判断

### スプリットプッシュすべき時
- 1v1で対面に勝てる場合
- チームが集団戦で不利な場合（ビハインド時のサイド圧力）
- オブジェクトが反対側にスポーンする時（ドラゴン時はトップ、バロン時はボット）
- 自分がスプリットの方がチームファイトより価値が高い場合
- TPまたは移動スキルで合流可能な場合

### グループすべき時
- チームが集団戦構成の場合（Ornn、Malphite等のエンゲージ構成）
- 味方が4v5を耐えられない場合
- 敵がハードエンゲージ構成で味方にディスエンゲージがない場合
- チームがリードしていてオブジェクトで勝負を決められる場合
- スプリット圧力をかけた後、敵が対応に分かれたタイミング（5v4の好機）

### ARAMの罠を避ける
- ミッドに5人集まり続けるのは非効率（XP分散、サイドウェーブ損失、圧力なし）
- 敵に強力なウェーブクリアがあるとミッドだけでは破れない
- **1-3-1配置**：3レーン同時にウェーブを押し、敵にウェーブクリアの選択を迫る
- **1-4配置**：スプリッター1人＋4人グループ。4人側がオブジェクト圧力、1人がサイド

### スプリットプッシュの鉄則
- ワードを設置して情報を確保（崩壊を防ぐ）
- 味方は必ずディスエンゲージ可能な位置取り。味方が開けた場所で4v5を始めるのが最大のミス
- 「敵が来たら逃げる、来なければタワーを折る」が基本

## 9. オブジェクトスポーン基準の意思決定タイムライン

### 90秒前：準備フェーズ
- オブジェクト周辺にワード設置開始
- チームにオブジェクトの計画を伝達（ピング）
- サイドウェーブの状態を確認し、スローブッシュを開始

### 60秒前：セットアップフェーズ
- ミッド＋隣接サイドレーンをプッシュ
- リコールが必要ならこのタイミング（45〜60秒前）
- 敵視界をスイーパーで除去開始
- コントロールワードをオブジェクト周辺に設置

### 30秒前：ポジショニングフェーズ
- チーム全員がオブジェクト周辺に集結
- 人数有利を確認（敵の位置把握）
- ピット周辺と進入路の視界を最終確認
- フランクルートにワード配置

### スポーン時：実行フェーズ
- スマイト保持者（JG）が確実にスマイト圏内で仕留める
- フェイスチェックせず視界で安全確認してから開始
- 敵全員の位置が不明なら無理に開始しない

### バロン特有の注意点
- ウェーブが全レーン押せていなければバロンを始めない
- 視界確保が不完全なら始めない
- チームのコミットが分かれている（一部が戦闘、一部がバロン殴り）状態を避ける
- この3つがバロンスロー（逆転される）の最大要因

## 10. ロール別マクロ責任

### トップレーン
- **ウェーブ管理が最重要スキル**：フリーズ、スローブッシュ、ファストプッシュを駆使
- **スプリットプッシュ判断**：TPがある時はスプリット→TP合流、TPがない時はグループを検討
- **テレポートの使い方**：ドラゴンファイト/バロンファイトへのTP合流 vs サイドレーン圧力継続の判断
- **リフトヘラルドへの参加**：JGと連携してヘラルド確保→タワープレート獲得
- **よくあるミス**：永遠にスプリットしてグループしない → TPタイマー/重要オブジェクトにはグループ

### ジャングル
- **パスイング判断**：ファーム vs ガンク vs オブジェクトの3択を常に判断
- **オブジェクトコントロール**：ドラゴン、ヘラルド、バロンの確保がJGの最大責任
- **ヘラルドの戦略的活用**：ドラゴンを1〜2体捨ててでもヘラルド2体確保→レーンをスノーボール
- **視界管理**：敵ジャングル深部へのワード設置、オブジェクト周辺の視界確保
- **スマイトファイト**：スマイトを確実に当てるためにHP管理。50/50を避ける
- **Mid-JGシナジー**：ミッドとの連携（スカトル、侵入、ガンク）が勝率に直結

### ミッドレーン
- **レーン優先権の確保**：ウェーブを先に押すことでリバーへの移動が先になる
- **ローミング**：ボット/トップへのローム＋JGとのスカーミッシュ参加
- **Mid-JGシナジー**：JGと連携してスカトル確保、敵JG侵入、2v2
- **ウェーブ管理**：オブジェクト前にミッドを押す責任。ミッド優先権=オブジェクトアクセス
- **マップの中心**：ミッドタワーが落ちるとマップ中央の支配権を失う。最重要タワー

### ADC（ボットレーンキャリー）
- **ファームとスケーリング**：安全にCSを回収し、アイテムスパイクに到達
- **タワー/オブジェクトDPS**：タワー、ドラゴン、バロンへのDPSがADCの主要マクロ価値
- **ファーストタワー後のローテーション**：ボットタワー破壊後、ミッドにローテーションして安全にファーム
- **ポジショニング**：集団戦では前に出すぎない。サイドウェーブ回収時はマップを確認
- **サイドウェーブ回収**：中盤以降、サイドウェーブを効率的に回収してゴールドリードを維持

### サポート
- **視界管理の主責任者**：ワード設置＋スイーパーによる視界否定がチームの目
- **ローミング**：レーン優先権確保後またはリコール後にミッド/JGへローム
- **オブジェクトセットアップ**：オブジェクト60〜90秒前から視界確保を開始
- **ショットコール**：プロシーンではサポートがショットコーラーであることが多い。チームの動きを指示
- **レーニング後の移動**：ファーストタワー崩壊後、ADCに張り付くのをやめ、最も強い味方をサポート
- **Lv3後のオブジェクト参加**：リコール後Lv3でオブジェクトに向かい、JGを支援

## 判断フレームワーク：状況別クイックリファレンス

### 「今何をすべきか？」の判断ツリー

```
1. オブジェクトが60秒以内にスポーンするか？
   → YES: オブジェクト準備（視界、ウェーブプッシュ、リコール判断）
   → NO: 次へ

2. 敵がデスしているか？（人数有利）
   → YES: 最も価値の高いオブジェクトを取る（タワー > ドラゴン/バロン > ジャングル侵入）
   → NO: 次へ

3. アイテムスパイク分のゴールドがあるか？
   → YES: ウェーブをクラッシュしてリコール
   → NO: 次へ

4. サイドウェーブが押されてきているか？
   → YES: 回収に行く（ただしオブジェクト直前なら無視も検討）
   → NO: 次へ

5. レーン優先権はあるか？
   → YES: ローム/視界確保/ジャングル侵入
   → NO: ウェーブを管理して優先権を確立

6. 敵の位置は把握できているか？
   → YES: 情報に基づいてプレイ（キャッチ、オブジェクト、スプリット）
   → NO: 安全にプレイし、視界確保を優先
```

### ゴールド差別の行動指針

| ゴールド差 | 推奨行動 |
|-----------|---------|
| +3000G以上 | オブジェクトを積極的に強制。視界支配でマップを制圧。敵ジャングル侵入 |
| +1000〜3000G | アイテムスパイクを活かしてオブジェクトコンテスト。有利な戦闘を選ぶ |
| ±1000G | 五分。ウェーブ管理と視界で小さなアドバンテージを積み重ねる |
| -1000〜3000G | 無理な戦闘を避ける。クロスマップトレード。パワースパイクを待つ |
| -3000G以上 | タワー下でファーム。オブジェクトは味方構成で勝てる場合のみコンテスト。スプリット圧力検討 |

### ゲーム時間別の重点

| 時間帯 | 重点 |
|-------|------|
| 0〜5分 | レーニング、ウェーブ管理、JGパス把握 |
| 5〜10分 | ファーストドラゴン/ヘラルド、ガンクタイミング、プレート |
| 10〜14分 | ヘラルド2体目、タワープレート、ローム |
| 14〜20分 | ドラゴンスタック、タワー交換、中盤パワースパイク |
